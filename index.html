Think in nodes, not threads

Every node is a thought. Every connection is context. Tesserite reads the graph upstream from your prompt — the LLM sees why you're asking, not just what.

Branch freely. Mask what you don't need. Steer conversation flow with diagenic controls overlaid right on the node. A new interaction paradigm for human-LLM communication.

Your canvas has a z-axis

Semantic clustering arranges nodes by meaning. Timelines rise along a temporal axis. Ground maps anchor abstract thinking to spatial metaphors.

Import .glb models. Export interactive 3D HTML views anyone can explore — no Obsidian required.

Context with surgical precision

Mask irrelevant branches. Weight prompts over the context chain. Choose which direction new responses flow.

Visual prompt debugging — see exactly what the model reads, catch when irrelevant context pollutes responses. The canvas is the interface and the prompt engineering tool.

Everything else you'd want, and a few things you didn't know you needed

Bring Your Own Models

Claude, Gemini, OpenAI via OpenRouter, local via Ollama. Switch mid-conversation. Compare outputs. Batch mode fires off prompts in parallel.

Search Grounding

Brave Search and Exa inject live web results directly into your LLM's context. Labeled, sourced, and connected to the response node.

RAG Over Your Vault

Semantic search finds relevant notes you forgot you wrote. Boosts human-written content. Your knowledge stays central, not the model's.

Personas & Dialogue

Distinct LLM personas with their own system prompts, colors, and persistent memory. Structured hashtag channels for technical review, ideation, feedback.

Canvas Agent

The LLM manipulates the canvas itself — creating nodes, drawing edges, moving things around. Sandboxed, audited, and configurable.

Templates & Export

Reusable node structures. Diff mode. 3D HTML export. Godot game engine integration. YouTube and audio transcription built in.

Who is Tesserite for?

Visual thinkers. Researchers. Worldbuilders. Maximalists. Anyone whose ideas have topology.

Tesserite meets neurodivergent thinkers where their thinking already happens — designers, 3D artists, filmmakers, and the millions of creative people now navigating technical systems through vibe coding who need to see what they're building.

Spatial interfaces lower the floor and raise the ceiling. Basic use becomes more intuitive. Advanced use becomes more powerful.
